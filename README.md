# 📸 Photography Portfolio Website

A responsive and modern Photography Portfolio Website developed using **HTML, CSS, and JavaScript**. This project showcases an event photographer's work, services, and contact information through a clean and user-friendly interface.

## ✨ Features

- 🏠 Attractive Home Page
- 🖼️ Gallery with image zoom effect
- 👤 Professional About Page
- 💼 Services displayed using cards
- 📞 Contact page with contact form
- 🔐 Login and Registration pages (Frontend)
- 📱 Responsive and user-friendly design
- 🧭 Common navigation bar and footer across all pages

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open `home.html` in any web browser.

No additional software or installation is required.

## 🎯 Purpose

The purpose of this project is to create a professional photography portfolio website that allows photographers to showcase their work, services, and contact information through a modern web interface.
